## ⚙️ Requirements
- Arch Linux
- Wayland
- Hyprland
- waybar
- dunst
- kitty (or preferred terminal)

---

## 🚀 Install (manual)

Clone:
git clone https://github.com/Show0ne/dotfiles

Copy configs:
cp -r .config/* ~/.config/

---

## ⚡ Optional (recommended)

Create symlinks instead of copying:
ln -s ~/dotfiles/.config/hypr ~/.config/hypr
